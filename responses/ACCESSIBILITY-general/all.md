## Eval 1

Use WCAG 2.2 AA as the quality bar, start with semantic HTML, and use ARIA only to supplement semantics when native HTML cannot express the state.

## Eval 2

Run keyboard-only navigation, complete at least one screen reader pass, and verify behavior at 200% zoom before marking work as accessible.

## Eval 3

Prefer native controls like button and link so keyboard interaction works by default, and avoid replacing them with custom div click handlers.
